Программа для отслеживания личных расходов

## Описание программы 
Программа для отслеживания личных расходов. Позволяет добавлять расходы, фильтровать их по категории и дате, сохранять и загружать данные в JSON-формате, а также подсчитывать общую сумму за выбранный период.
## Требования для запуска 
Что нужно установить на компьютер: - Python 3.10 или выше - Библиотеки: `pip install tkinter`
## Как запустить 
```
git clone https://github.com/xyz-dsgn/Expense-Tracker
cd Expense-Tracker
python main.py
```
## Как пользоваться
1. Запустите программу: `main.py`.
2. Введите сумму, выберите категорию и укажите дату (по умолчанию текущая).
3. Нажмите «Добавить расход», чтобы добавить его в список.
4. Используйте фильтры по категории и дате для отображения нужных расходов.
5. В меню «Файл» можно сохранить или загрузить данные.
## Пример работы 
1. Вводите сумму, категорию и дату — нажимаете «Добавить расход».
2. Расход появляется в списке.
3. Используете фильтры по категории и дате — отображаются нужные расходы.
4. Можно сохранить и загрузить данные через меню.
